# @react-native/material-symbols

Material Symbols for React Native

## Installation

```sh
npm install @react-native/material-symbols
```

## Usage

```js
import { MaterialSymbolsView } from "@react-native/material-symbols";

// ...

<MaterialSymbolsView color="tomato" />
```
